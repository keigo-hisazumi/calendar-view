# カレンダーアプリ - 予定共有

他人と予定を共有できるカレンダーアプリケーションです。Vue.js 3とViteで構築されたモダンなWebアプリケーションで、将来的にはSpring Bootとの連携を想定しています。

## 🎯 主な機能

- **📅 月間カレンダー表示**: 直感的なカレンダーインターフェース
- **➕ 予定の追加・編集・削除**: 簡単に予定を管理
- **👥 予定の共有**: 他のユーザーと予定を共有可能
- **📱 レスポンシブデザイン**: モバイルデバイス（iOS/Android）に対応
- **💾 ローカルストレージ**: ブラウザにデータを保存

## 🚀 セットアップ

### 必要な環境

- Node.js (v18以上推奨)
- npm または yarn

### インストール

```bash
# 依存関係のインストール
npm install

# 開発サーバーの起動
npm run dev

# ビルド
npm run build

# プレビュー
npm run preview
```

## 📖 使い方

1. **カレンダーの閲覧**: 月間カレンダーで予定を確認
2. **日付の選択**: カレンダーの日付をクリックして選択
3. **予定の追加**: 「新しい予定」ボタンをクリックして予定を追加
4. **予定の共有**: チェックボックスを有効にして共有ユーザーを選択
5. **予定の編集・削除**: 予定リストから編集または削除

## 🛠️ 技術スタック

- **フロントエンド**: Vue.js 3 (Composition API)
- **ビルドツール**: Vite
- **スタイル**: Scoped CSS
- **データ保存**: LocalStorage（モック）

## 📱 モバイル対応

このアプリケーションはレスポンシブデザインを採用しており、スマートフォンやタブレットでも快適に使用できます。将来的にはCapacitorなどを使用してネイティブアプリとして配信予定です。

## 📝 モックアプリについて

このバージョンはモックアプリケーションとして作成されており、以下の点に注意してください：

- データはブラウザのローカルストレージに保存されます
- ユーザー認証は実装されていません
- 実際の共有機能は実装されていません（UIのみ）
- サーバーとの通信は行われません
